⚙️ How to Run the Project

This template uses Vite as the module bundler. Follow these steps to run the project locally:

  1. Make sure Node.js and npm are installed.
  2. Open your terminal and navigate to the project folder.
  3. Install dependencies: 
    npm install
  4. Run the development server: 
    npm run dev
  5. The template will be available at http://localhost:5173.

🖼️ How to Change Images

All images and media files are located in the /public/images, /public/icons, and /public/video folders. You can replace them with your own files and update the paths in index.html. 

<img src="/images/hero.jpg" alt="Hero Image">

📝 How to Edit Text

To change any text, open index.html and update the content directly. For example: 

<h1 class="text-4xl font-bold">Main Heading</h1>

🔗 Logo Link to Homepage

Make sure the logo in the header links to the homepage. Example:

<a href="index.html">
  <img src="/icons/logo.png" alt="Logo">
</a>
